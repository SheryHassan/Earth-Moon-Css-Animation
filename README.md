# Earth-Moon-Css-Animation
# 🌍 Earth & Moon CSS Animation

A simple front-end project that simulates the **moon orbiting around the earth** using **pure HTML and CSS**.

This project was built as a practice exercise while learning **CSS animations, transforms, and positioning**.

---

## 🌐 Live Demo

🚀 **[View Live Project](https://sheryhassan.github.io/Earth-Moon-Css-Animation/)**

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* CSS Animations
* CSS Transforms

⚡ **No JavaScript was used in this project.**

---

## ✨ Features

* Smooth moon orbit animation
* Two different techniques to simulate orbital motion
* Animation speed changes on hover
* Clean and minimal layout
* Built using only HTML and CSS

---

## 🎯 Learning Objectives

This project was created to practice:

* CSS `position`
* CSS `transform`
* CSS `animation`
* `@keyframes`
* Creating orbital motion effects
* Combining `rotate()` and `translate()` in CSS

---

## 🔄 Animation Techniques

### Method 1

Using a **rotating container** that orbits around the earth.

### Method 2

Using **rotate + translate** together to simulate the moon orbit.

---

## 🎮 Interaction

When hovering over the earth container, the animation **slows down**.
